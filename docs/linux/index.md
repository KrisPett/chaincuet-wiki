### Linux Commands

##### Change owner of a folders recursively

```
sudo chown -R work ./
```

#### Check OS

```
cat /etc/os-release
```

#### List users

```
cat /etc/passwd
```

#### Change password

```
sudo passwd <username>
```

#### Check storage

```
df -h 
sudo fdisk -l
```
